UnityCindy3D

UnityCindy3D is an online 3D drawing and game programming environment using CindyJS.

Supported Browsers

Desktop browsers (64bit) :

  • Firefox 65+ (recommended)
  • Chrome 72+ (*)
  • Safari 12+
  • Edge 42+

* The code opened from local file URLs doesn't work on Chome for the security reasons.

Desktop browsers (32 bit) :

not supported

Mobile browsers :

(currently) not supported

Usage

  • Drag the 3D viewer to rotate the camera.
  • Click the 3D viewer to give key focus to the viewer.
  • Click the 3D viewer and press F4 key to enter/exit the test mode.
    (test mode)
    s key: show/hide a sphere
    t key: show/hide a torus
    b key: show/hide a flying ball
